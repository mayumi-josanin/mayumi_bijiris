# Mayumi Bijiris Survey App

アンケート回答アプリとアンケート管理アプリをまとめた静的 Web アプリです。

## ファイル

- `index.html`: アプリ本体
- `styles.css`: 画面デザイン
- `app.js`: 回答、履歴、管理、アンケート作成機能

## 使い方

1. `index.html` をブラウザで開きます。
2. お客様は「回答アプリ」でお客様情報を保存してからアンケートに回答します。
3. 管理者は「管理アプリ」からログインします。

管理者ログイン:

- ID: `admin`
- PASS: `admin123`

## データ保存

この版は GitHub Pages で公開できる静的アプリとして作成しています。回答データ、アンケート、ログイン状態はブラウザの `localStorage` に保存されます。

複数端末の回答をサーバーで一元管理する場合は、Firebase、Supabase、独自 API などのバックエンド接続を追加してください。

## デプロイ

GitHub Pages は `main` ブランチ直下から公開しています。ファイルを編集した後は、次のコマンドで公開版に反映できます。

```bash
./deploy.sh "変更内容のメモ"
```
